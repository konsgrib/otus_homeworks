Домашнее задание #1

Программа собирает ссылки из подраздела сайта www.ss.com.
Заходит по собранным ссылкам и собирает ссылки на дополнительные страницы определённого класса.
Чтобы сохранить ссылки в файл необходимо задать значение переменной SAVE_TO_FILE в True, иначе ссылки будут просто выводиться в терминал.
Для запуска программы необходимо установить зависимости указанные в requirements.txt и запустить файл main.py командой python main.py
